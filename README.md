# GamePath 游戏攻略站

这是一个最小可行版本的游戏攻略网站，适合先快速上线到 GitHub Pages。

## 当前功能

- 首页展示攻略列表
- 点击卡片查看攻略详情
- 支持关键词搜索
- 不需要后端，直接用静态文件部署

## 本地查看

直接双击 `index.html` 就能打开页面。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个公开仓库，例如 `gamepath-guides`
2. 把这个项目代码推送到那个仓库
3. 在 GitHub 仓库里打开 `Settings` -> `Pages`
4. 在 `Build and deployment` 里选择 `Deploy from a branch`
5. Branch 选择 `master`，文件夹选择 `/ (root)`
6. 保存后等待几十秒到几分钟，GitHub 会给你一个公开网址

## 后续最适合新增的内容
